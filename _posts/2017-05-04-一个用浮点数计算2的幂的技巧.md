## 适用范围

仅适用于计算$$2^n$$的精确值，且$$\left\vert n\right\vert<2^{14}$$

浮点数能精确表示$$2^n$$，因为大部分浮点数内部都以2为底数，n的范围与浮点数类型有关。常用浮点数最高精度的long double/extended也只有15位阶码。

## 使用方法

直接使用pow函数即可计算。

### n>0

#### C++

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	cin>>n;
	cout.precision(0);
	cout<<fixed<<pow(2.0L,n)<<endl;
	return 0;
}
```

#### C

```c
#include<stdio.h>
#include<math.h>
int main(void)
{
	int n;
	scanf("%d",&n);
	printf("%.0Lf\n",powl(2.0L,n));
	return 0;
}
```
